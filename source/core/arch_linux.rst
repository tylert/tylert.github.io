Pacman
------

::

    # -i = --info
    # -S = --sync

    pacman -Si foo  # show info about available package 'foo'
    pacman -Ss bar  # show names of available packages with 'bar' in the name

    pacman -Qi foo  # show info about installed package 'foo'
